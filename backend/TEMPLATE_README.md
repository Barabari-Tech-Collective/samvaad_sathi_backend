# Template Archive

This repository originally started from a generic FastAPI backend template. The old template documentation has been removed from active guidance because it referenced modules, routes, CI assumptions, and setup steps that are no longer part of Samvaad Sathi.

Do not use this file for setup or architecture decisions.

Use these current docs instead:

- `backend/backend/README.md`: main backend overview and API surface
- `backend/docs/DEV_GUIDE.md`: local development workflow and rules
- `backend/docs/architecture.md`: current architecture
- `backend/docs/SOURCE_FILE_GUIDE.md`: source-file guide

Deprecated template concepts that are not current project guidance:

- `Account` model and account-management endpoints
- Template CI badges, Codecov setup, CODEOWNERS setup, and semantic commit workflow
- Generic pyenv/pre-commit instructions as required setup
- Docker ports and service names from the template
